Emails Input

An easy way to input emails on your app!

This is the example page for this lib. Here it's documented how to use each option and members of the class and its intances.

Basic Example
EmailsInput(node);

With the basic example, you should be able to write new emails (separated by comma) or by pasting them. Once an email is created, you can remove it.

It's possible to count the valid e-mails by clicking on "get emails count". And if you want to spicy things up, a random email can be added by clickin on "add email".

Pre Filled With Emails

An EmailsInput instance accepts a list of emails to prefill the editor with them.

var emails = ['john@miro.com', 'invalid.email', 'mike@miro.com', 'alexander@miro.com']; var input = EmailsInput(node, { emails });

It's also possible to set the emails for the instance later. This is used in the reset button below.

input.setEmails(emails);
